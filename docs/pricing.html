Frequently Asked Questions

Where does my data go?

Nowhere. Cerberus runs entirely inside your VPC. Tool call content, PII, and detection results never leave your infrastructure. The only external call is a startup license validation (key → our API, no payload data).

How does the license work?

After payment, you receive a license key by email (format: cbr_ent_…). Set it in your .env file. The gateway validates it against our API on startup — hard-fail if invalid, grace mode on network errors so an outage never blocks your production system.

What happens when my license expires?

You receive expiry warnings at 30 days. After expiry, the gateway exits on next restart. Your running containers continue until they're restarted, giving you time to renew without a hard outage.

Is the core detection engine open source?

Yes. @cerberus-ai/core is MIT licensed and published on npm. The enterprise product adds the Gateway, monitoring stack, and production deployment tooling on top of the open-source core.

Can I try before I buy?

Yes — try the live attack demo and install the SDK via npm install @cerberus-ai/core. Both are free. Contact us if you'd like a private demo call.
