How It Works

4-layer detection engine that correlates signals across the entire agent session.

L1

Data Source Classifier

Scans tool results for PII (emails, SSNs, phone numbers) and tracks which tools access privileged data.

L2

Token Provenance Tagger

Tags content from untrusted external sources. Tracks token count and origin so the engine knows what's tainted.

L3

Outbound Intent Classifier

Checks if outbound tool calls contain PII from session state. Catches exfiltration attempts in real time.

L4

Memory Contamination Graph

Tracks contamination across sessions. Detects delayed attacks where poison is written now and exploited later.

CE

Correlation Engine

Aggregates all layer signals into a 4-bit risk vector and score. Maps score to action: log, alert, or interrupt.
